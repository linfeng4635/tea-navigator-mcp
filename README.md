# tea-navigator-mcp (地缘名茶导航服务)

一个基于 Model Context Protocol (MCP) 的标准空间地理定位智能工具服务端。该服务专为大模型端（如聆思智能硬件、AI 智能体）设计，能够动态解析用户输入的当前地理位置描述，精准匹配中国各省份核心产茶区的非遗名茶数据，并输出包含风味特点、专家级冲泡建议的富文本指南。

## 🚀 项目特性

* **数据逻辑解耦**：采用结构化的 `tea_data.json` 存储名茶知识，业务逻辑通过 `main.py` 动态加载，支持无缝扩容。
* **智能位置泛化**：不仅支持标准省份名称，还能模糊识别如“我在西湖区区政府附近”、“出差到了大理”等复杂的端侧 GPS 逆地理编码或口语化文本。
* **全方位冲泡指南**：不仅推送茶叶，还针对不同茶类的特性（如碧螺春的娇嫩、单丛的耐泡）给出水温（如 85°C、100°C）、茶具及投茶手法的精准指导。
* **标准原生接入**：基于 Anthropic 官方 `fastmcp` 框架构建，完美兼容魔搭 MCP 广场的 `STDIO` 托管与 `SSE` 代理。

## 🛠️ 核心工具声明 (Tools)

魔搭平台与大模型（LLM）将通过以下定义自动注册并调用本工具：

### `push_local_tea_info`

根据用户当前所处的地址、省份或城市，智能匹配并推送当地最具代表性的特色茶叶信息。

* **输入参数 (Arguments)**:
    * `address` (string, **必须**): 用户的当前地理位置描述或定位文本。
    * *示例输入*：`"我现在在福建武夷山景区"` 或 `"杭州市余杭区"`
* **输出结果 (Returns)**:
    * `string`: Markdown 格式的当地名茶风味科普与冲泡指南。若未匹配到专属茶区，将自动输出万能待客茶（传统茉莉花茶）作为优雅兜底。
